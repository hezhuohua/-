方法2：手动启动

步骤2：启动前端服务（新终端）

python -m http.server 8080

🛠️ 故障排除

常见问题及解决方案：

  • Python命令不可用：确保Python已安装并添加到系统PATH
  • 端口被占用：检查8000和8080端口是否被其他程序占用
  • 依赖缺失：运行 pip install fastapi uvicorn
  • 权限问题：以管理员身份运行命令提示符

检查端口占用：

netstat -ano | findstr :8000
netstat -ano | findstr :8080
