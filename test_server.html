⚠️ 当前状态

🔧 服务测试

🛠️ 故障排除

  1. 后端服务未启动：运行 python backend/minimal_server.py
  2. 端口被占用：检查8000和8080端口是否被其他程序占用
  3. 依赖缺失：运行 pip install fastapi uvicorn
  4. Python环境问题：确保Python 3.7+已正确安装

✅ 快速启动指南

方法1 - 使用启动脚本：

python start_services.py

方法2 - 手动启动：

# 终端1 - 启动后端
python backend/minimal_server.py

# 终端2 - 启动前端（可选）
python -m http.server 8080
